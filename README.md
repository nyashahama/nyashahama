## About Me

Backend-heavy full-stack engineer from **Johannesburg, South Africa** who ships production-grade systems — not prototypes. I care about clean architecture, raw performance, and building products that solve real problems, especially across African markets.

- Currently building **event-driven backends** in Go with PostgreSQL, Redis & NATS
- I write TypeScript/Next.js frontends and reach for C++ when performance is non-negotiable
- Experienced with HIPAA/POPIA-compliant architecture and high-throughput auth systems
- Open to roles where the work actually matters
